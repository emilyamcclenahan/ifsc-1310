Progressive Enhancement

My Thoughts

In my opinion, Progressive Enhancement is what allows for any website, browser, or internet connection to get the best functionability across ALL websites, browsers, & internet connections. It ensures stability by giving us basic user experience & cross compatibility.

The Modern Web is forever changing. It is very important that all past & future websites are easily accessible & run well across multiple different browsers & networks. Basic content & basic functionality should be accessible to all web browsers.

Using this method will save time in the end on projects. It will syncronize the data across all the web browsers so that you're not having to go check each browser(i.e. Mozilla, Firefox, Microsoft Edge....) to make sure that all your information is readable & accessible.
